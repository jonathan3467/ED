##### Funcionamiento #####
para este proyecto, tenemos varios auxiliares, pero solamente necesitaremos 4
algoritmos clave, que serian "frecuenciaLetras" que como su nombre indica es
para saber la frecuencia de cada letra, donde le tenemos que pasar una palabra
y esta nos va a regresar una lista de caracteres, el primer caracter que este
en la lista es el que más se repite, van de mayor a menor frecuencia, hay dos
uno donde te va a regresar con numero cuantas veces se repite cada eltra, y
otro donde te va a regresa solamente las letras ordenadas

Luego tenemos el más complicadillo el de "construirHuffman" que como su nombre
indica construimos el arbol de Huffman segun lo visto en clase, donde los
subarboles izquierdos son 0 y los subarboles derechos son 1 y tienen las letras
solamente el lado derecho, para este utilizaremos lo que te de el codigo
anterior y te construira un arbol.

Para el siguiente que es "codigos" a este le tenemos que psar el arbol del
anterior codigo que nos de, y este nos dara el codigo de cada eltra, si es 1,
o si es 000001, y esto nos ayudara para el siguiente

Para el ultimo llamado "decodificar" le tendremos que psar el arbol que nos
dio el codigo 2, y tendremos que pasarle ademas el codigo que creemos hace
la palabra, en los codigos hay un ejemplo para que sea más facil y no tengas
que escribir muchos 0, y este al final si pusiste todo bien te dara la palabra
