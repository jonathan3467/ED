##### Objetivo #####
Desde mi punto de vista el objetivo de este proyecto, es implementar todo lo
que hemos visto hasta ahorita, desde checar cuantas veces se repite una letra
hasta el tema de arboles que fue el ultimo que vimos, y ponernos a pensar las
soluciones de esta

##### Funcionamiento #####
para este proyecto, tenemos varios auxiliares, pero solamente necesitaremos 4
algoritmos clave, que serian "frecuenciaLetras" que como su nombre indica es
para saber la frecuencia de cada letra, donde le tenemos que pasar una palabra
y esta nos va a regresar una lista de caracteres, el primer caracter que este en la lista es el que más se repite, van de mayor a menor frecuencia

Luego tenemos el más complicadillo el de "construirHuffman" que como su nombre
indica construimos el arbol de Huffman segun lo visto en clase, donde los
subarboles izquierdos son 0 y los subarboles derechos son 1 y tienen las letras
solamente el lado derecho
